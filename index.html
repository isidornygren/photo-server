Welcome to the image server

Use the url as a path traversal, and add image transform parameters to the query parameters.

URL Query parameters

width height dither
u32 u32 EPaperSeven

So for example, in order to send a query for a dithered image with a width and a height, you would query

http://localhost:8080/path/to/image.jpg?width=100&height=200&dither=EPaperSeven
